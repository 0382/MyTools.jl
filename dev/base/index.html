向量

MyTools.Vec — Type
struct Vec{N, T<:Real} <: AbstractVector{T}
const Vec2{T} = Vec{2, T}
const Vec3{T} = Vec{3, T}
const Vec4{T} = Vec{4, T}

笛卡尔坐标系向量。

source
  • 定义了基本的向量加减法、数乘（除）
  • 实现了AbstractVector的接口，引入LinearAlgebra即可进行内外积操作
  • 对于维度小于等于4的向量，可以使用.x, .y, .z, .w索引取值（不可更改）
